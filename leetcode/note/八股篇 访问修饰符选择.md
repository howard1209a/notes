# 访问修饰符选择

平时在写代码时，属性、静态属性、方法、静态方法如何选择访问修饰符呢？

### 对于静态方法和静态属性

静态方法和静态属性一般都是工具类，我们关心的是可以在哪调用的问题。

- private：希望只能在本类调用
- default：希望只能在本包调用
- protected：希望子类都可以调用
- public：希望任何地方都可以调用

### 对于方法和属性

我们关心的是这个方法或者属性的可见性问题。

- private：希望仅本类可见
- default：希望本包的对象或子类都可见
- protected：希望子类无论在哪都可见
- public：希望任何包的对象或子类都可见
