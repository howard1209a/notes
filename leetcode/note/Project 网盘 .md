# 网盘

## 设计实现

### 功能列表

- 前后端分离，搭好项目架子
- 用户注册登陆
- 上传图片，图片列表
- 个人相册浏览，图像详细资料展示卡片
- 图片分级保管
- 图片链接
- 回收站
- 图库浏览
- 图库浏览衍生功能
  - 点赞
  - 评论
  - 分享
  - 收藏
- 个人仪表盘监控
- 图片复杂检索（个人/全站）
- 积分
- 下单（下单超时未支付）

### 微服务划分

- user-service\*1

### Mysql 表设计

- 用户表
  - 用户 ID
  - 用户名
  - 密码
  - 邮箱
  - 创建时间
  - 修改时间
- 图片
  - 图片 ID
  - 用户 ID
  - 图片磁盘路径
  - 上传时间
  - 描述
- 标签表
  - 标签 ID
  - 标签名
